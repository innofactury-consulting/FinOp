# FinOp™ — Financial Operations Engine
### by InnoFactury Consulting LLC

Motor de orçamento indutivo baseado em drivers operacionais para grandes operações industriais (Mining, Automotive, Papel & Celulose).

**Acesse:** [finop.innofactury.com](https://finop.innofactury.com)

---

## Funcionalidades

- 🏢 Configuração por empresa com parâmetros macroeconômicos dinâmicos
- 🗂 Centros de Custo por departamento/planta
- 📋 Tipos de Despesa OPEX/CAPEX com motor de correlação
- ⌨ Lançamento de Drivers Operacionais → Cálculo automático
- 📊 Orçamento Consolidado com narrativa "Produzir Mais com Menos"
- ♻ Projetos Lean/Kaizen com abatimento automático de OPEX
- 🏗 CAPEX & ROI cruzado (MTBF, OPEX, Risco Laboral)

## Tecnologia

- Single Page Application (HTML + CSS + JS)
- Chart.js para visualizações
- Deploy: GitHub Pages + Custom Domain
- Sem dependências de backend (fase 1)
